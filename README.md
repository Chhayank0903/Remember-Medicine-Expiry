# Remember-Medicine-Expiry
An AI-powered medicine expiry tracker that alerts rural health workers and families before it's too late.

MediGuard is a smart, simple, and mobile-friendly web application designed for rural India to solve one critical problem — expired medicines being used unknowingly by families, elderly people, and health workers.
In rural areas, medicines are often stored for long periods at homes, Anganwadi centres, Primary Health Centres, and kirana stores without anyone regularly checking their expiry dates. This leads to serious health risks especially for children and elderly people.
MediGuard solves this by:

📷 Scanning medicine packs using camera and reading expiry dates automatically using OCR technology
🔴 Instantly showing whether a medicine is Expired, 🟡 Expiring Soon, or 🟢 Safe to use
🔔 Sending automatic alerts before a medicine expires so action can be taken in time
🗣️ Reading the result aloud in simple English for low-literacy users
📊 Generating reports for ASHA workers to share with supervisors
💬 Allowing users to chat with an AI agent and ask questions like "Which medicines expire this week?"
